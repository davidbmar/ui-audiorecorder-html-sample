Troubleshooting

Common issues and solutions

âŒ Microphone permission denied

âš ï¸
Solution: Make sure you're serving the page over HTTPS (or localhost). Modern browsers require secure context for microphone access.

âŒ No audio in recordings

  • Check that your microphone is working in other apps
  • Look for browser notifications asking for microphone permission
  • Try refreshing the page and granting permission again
  • Check the browser console for error messages

âŒ Recordings not saving

  • Check that IndexedDB is supported (all modern browsers)
  • Clear browser data and try again
  • Check the storage status indicator in the control panel
  • Look for console errors related to storage

âŒ Files not loading

  • Make sure all three files (sai.css, storage.js, app.js) are in the same directory
  • Check the browser console for 404 errors
  • Verify file paths in your HTML are correct
