Integration Checklist

Make sure you've got everything set up correctly

üìã
Use this checklist to ensure your integration is working properly.

Before Integration

  • ‚úÖ Downloaded sai.css, storage.js, and app.js files
  • ‚úÖ Verified files are accessible from your HTML page
  • ‚úÖ Tested on HTTPS or localhost (required for microphone access)
  • ‚úÖ Checked browser compatibility (modern browsers only)

After Integration

  • ‚úÖ CSS styles are loading correctly
  • ‚úÖ JavaScript files load without errors (check console)
  • ‚úÖ Required HTML elements have correct IDs
  • ‚úÖ Microphone permission is granted when recording starts
  • ‚úÖ Audio chunks appear after recording
  • ‚úÖ Waveform visualizations render properly
  • ‚úÖ IndexedDB storage is working (check storage status)

Testing Your Integration

  1. Click the record button and grant microphone permission
  2. Speak for a few seconds and click stop
  3. Verify audio chunk appears with waveform
  4. Click the waveform to test seeking
  5. Refresh the page and check if recording persists
  6. Test on mobile device for responsive behavior
