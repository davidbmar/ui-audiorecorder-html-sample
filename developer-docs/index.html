Quick Start

Add professional audio recording to your app in 30 seconds

💡
No dependencies! Everything you need is included. Just copy the files and you're ready to go.

Step 1: Copy the files

Download and include these three files in your project:

File Structure
your-project/ ├── sai.css # SAI theme styles ├── storage.js # IndexedDB storage manager └── app.js # Core recording functionality

Step 2: Add to your HTML

Include the files and add the recorder HTML:

HTML
<!DOCTYPE html> <html lang="en"> <head> <meta charset="utf-8"> <meta name="viewport" content="width=device-width, initial-scale=1"> <link rel="stylesheet" href="sai.css"> </head> <body> <div class="app-shell"> <h1>My Audio Recorder</h1> <!-- Control Panel --> <div class="control-panel"> <div class="row"> <button id="btn-start" class="btn btn-accent">🎙 Start Recording</button> <button id="btn-stop" class="btn" disabled>⏹ Stop</button> <button id="clearBtn" class="btn">🗑️ Clear</button> </div> </div> <!-- Recorded Chunks --> <div class="chunks-panel"> <h3>Recorded Chunks</h3> <div id="chunkList" class="list"></div> </div> </div> <script src="storage.js"></script> <script src="app.js"></script> </body> </html>

Step 3: That's it!

Open your HTML file in a browser and start recording. The recorder will:

  • Request microphone permission
  • Record in gap-free chunks with configurable overlap
  • Display beautiful waveforms for each chunk
  • Save everything to IndexedDB for persistence
  • Handle browser refreshes gracefully
✅
Pro tip: The recorder automatically loads previous recordings when you refresh the page. No data loss!

Configuration Options

Customize the recorder to fit your needs

Option Default Description
chunkDuration 5000 Length of each recording chunk in milliseconds
overlapDuration 500 Overlap between chunks to eliminate gaps
mimeType 'auto' Audio format: 'audio/webm', 'audio/ogg', or 'auto'
enableStorage true Enable IndexedDB storage for persistence
enableWaveforms true Show waveform visualizations
theme 'sai' Visual theme: 'sai', 'minimal', or 'custom'

Example Configuration

JavaScript
const recorder = new SAIRecorder({ chunkDuration: 3000, // 3-second chunks overlapDuration: 300, // 300ms overlap mimeType: 'audio/webm', // Force WebM format enableStorage: true, // Save to IndexedDB enableWaveforms: true, // Show visualizations theme: 'sai', // Use SAI theme // Custom callbacks onRecordingStart: () => console.log('Recording started'), onChunkReady: (chunk) => console.log('New chunk:', chunk), onRecordingComplete: (session) => console.log('Session complete:', session) });
